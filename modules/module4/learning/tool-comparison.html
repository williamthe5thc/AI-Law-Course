Assignment 1: Tool Comparison Matrix

    Module 4 - Assignment 1 

For Assignment 1, you will be exploring the media generation capabilities of different AI platforms in different areas. For this assignment select and analyze three different AI media creation tools from different categories:

  • Image generation (e.g., DALL-E 3, Midjourney, Stable Diffusion)
  • Video creation (e.g., RunwayML, Synthesia)
  • Audio generation (e.g., Mubert, Soundraw)

You will be documenting the capabilities of different media generation tools and complete the reflection questions for each of the 3 types of media. After you have completed the exploration for each of the 3 types of media generation, complete the exploration reflection in the Module 4 Discussion area.
