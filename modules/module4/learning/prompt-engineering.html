Assignment 2: Prompt Engineering & Media Creation

    Module 4 - Assignment 2 

For this assignment, you will be creating media based on the assigned criteria for each of the 3 media types. You will be documenting your prompts, revisions of prompts, and all the outputs you generate as well as creating a brief explanation of how to create quality prompts for each media type. Please complete the documentation questions for each of the 3 media types listed below.

Click Here for Assignment 2

Tips for Success

  • Save everything - Keep records of all prompt versions and outputs
  • Be systematic - Use a consistent approach to prompt iteration
  • Document your reasoning - Note why you made specific changes
  • Include failed attempts - Failed outputs provide valuable insights
  • Analyze patterns - Identify what makes certain prompts more effective

Key Concepts

  • Prompt Iteration: The systematic process of refining prompts for better results
  • Media-Specific Parameters: Different prompt elements needed for each media type
  • Documentation: Capturing the evolution of prompts and outputs
  • Platform Differences: How different tools interpret similar prompts
  • Refinement Strategy: Approaches to improving initial results

Image Creation Guide

Before you begin, remember:

  • Save all versions of prompts and outputs
  • Note what changed between versions
  • Record why changes were made
  • Screenshot or save all attempts

Step 1: Initial Image Creation

  1. Select an image generation platform (DALL-E, Midjourney, Stable Diffusion, etc.)
  2. Create each of the following images below, be sure to document each version of your created images. Please note, your images don't need to take up a full page, you can scale them down to shorten the document

Step 2: Prompt Iteration (Create 3 Images)

  1. First Image: Create an image of a Product/Object
    • Start with basic description
    • Add details about style, lighting, perspective
    • Document each version and prompt until satisfied
  2. Second Image: Create an image of a Scene/Environment
    • Begin with location description
    • Add atmosphere, time of day, weather
    • Record how each addition affects output
  3. Third Image: Create an image of an Abstract Concept
    • Start with emotion or concept
    • Add artistic style references
    • Note how metaphors affect results

Documentation Format (This will be added to the Digital Portfolio)

  1. Initial Prompt: (Paste the text of your initial prompt)
    1. Result:
    2. Iteration 1:
  2. What changed:
    1. Why:
    2. Result:

Repeat for Each Iteration

  1. Final Prompt:
    1. Final Result:
  2. Analysis: Write a brief explanation of how to write a quality prompt for image generation

Video Generation Guide (30 Second Video)

Step 1: Planning

  1. Choose your video subject
  2. Write initial concept (15-30 words)
  3. Break video down into scenes (If necessary)

Step 2: Prompt Development

  1. Start with basic scene descriptions
  2. Add:
    • Movement descriptions
    • Transitions
    • Style elements
    • Timing specifications

Step 3: Refinement

  1. Test initial prompt
  2. Document results
  3. Identify areas for improvement
  4. Refine prompt focusing on:
    • Clarity
    • Specific actions
    • Visual style
    • Timing

Documentation Format (This will be added to the Digital Portfolio)

  1. Initial Concept for video:
  2. Scene Breakdown of video:
  3. First Prompt:
    1. Result:
    2. Issues:
  4. Refined Prompt:
    1. Changes made:
    2. Reason for Changes
  5. Write a brief explanation of how to create a quality prompt for generating video

Audio Creation Guide (Background Music 30 Seconds)

Step 1: Initial Audio Generation

  1. Select audio generation platform
  2. Define your needs:
    • Style of music
    • Length
    • Mood
    • Purpose

Step 2: Prompt Development

  1. Write basic prompt describing:
    • Genre
    • Tempo
    • Instruments
    • Mood

Step 3: Refinement

  1. Test initial prompt
  2. Document results
  3. Refine based on:
    • Melody
    • Rhythm
    • Energy level
    • Emotional impact

Documentation Format (This will be added to the Digital Portfolio)

  1. Initial Requirements/Needs
  2. First Prompt
    1. Result
    2. Issues
  3. Refined Prompt
    1. Changes
    2. Impact
  4. Final Result
  5. Write a brief explanation of how to create a quality AI prompt for generating audio

Submission Instructions: Submit your completed documentation for all three media types (image, video, and audio) as a single document. Include screenshots of your images, links to your video and audio files, and all prompt iterations as specified in the documentation format.
